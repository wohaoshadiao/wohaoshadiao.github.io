# 基于树莓派的四旋翼无人机视觉导航系统及方法

[TOC]

经一键启动后全程自主闭环飞行不需人为参与，结合了超声波传感器与图像采集处理模块，融合了计算机视觉技术，实现了飞行器在一定高度上稳定循迹飞行.

# 控制系统组成

  • 超声波传感器，光流传感器
  • 6050陀螺仪以及加速度计(姿态参数与欧拉角转换)
  • 树莓派（上位机）

# 循迹实现的原理

  • 四旋翼定高悬停

无人机一键起飞之后，利用超声波传感器获取高度测量信息，经过高度补偿算法将修正得到的高度值输送给飞行控制器，飞行控制器根据得到的高度位置与用户预期设定的高度值进行比较得出误差，依据内部的增量式pid算法进行闭环控制飞行器的油门输出来控制四旋翼稳定的保持既定高度

  • 循迹

图像控制器根据采集到图像信息传送到上位机，上位机根据接受到的图像信息利用计算机开源视觉库将获取到的每一帧图像进行灰度二值化处理，并进行一系列的去噪，形态学运算，Canny边缘计算，概率霍夫变换操作得到一系列易于上位机识别处理特征的图像。根据处理得到图像判断出所需要循迹路线的左右边缘，并根据视觉库内置的math库计算出中心线的起始坐标，以及四旋翼相对于中心线的偏移量，将所计算得到的偏移量反馈给主控制器，主控制器根据pid算法来控制横滚，最终实现相对于中心线的偏移量为0，再控制四旋翼的俯仰操作即可实现向前飞。
